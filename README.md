# NexoLauncher

A high-performance Minecraft Java and Bedrock Edition launcher for Android, designed with a sleek custom purple UI and powered by an AI Smart Assistant.

## Key Features

- **Custom Purple Theme**: A unique, visually appealing user interface tailored for a premium experience.
- **AI Smart Assistant**:
  - **Graphics Auto-Tuning**: Automatically optimizes game settings for your device's hardware.
  - **Crash Detection**: Identifies causes of game crashes and suggests fixes.
  - **Battery Saver**: Manages resources to extend playtime on the go.
  - **Cache Cleaning**: One-tap removal of temporary files to free up storage.
  - **AI Chat**: Integrated assistant to answer questions and help with configuration.
- **Google Account Integration**: Seamlessly backup and restore your launcher settings using your Google account.
- **Versatile Display**: Full support for both Portrait and Landscape orientations.
- **Optimized Performance**: Built on a solid foundation to ensure smooth gameplay on Android devices.

## Building and Installation

NexoLauncher is developed using Android Studio. To build the project from source:

1. Clone the repository.
2. Open the project in **Android Studio**.
3. Sync the project with Gradle files.
4. Use the build tasks in Android Studio to generate the APK.
